# Simon
Esto es un proyecto en el que voy a hacer un Juego llamado Simon, que va de seguir una secuencia de colores, en este proyecto estare aprendiendo manejar Arrays, API's ya que gracias a estas se podra jugar a este juego por voz


### 1. Redaccion del proyecto:

1. Utilizare dos arrays los cuales se tienen que ir comparando, uno del juego y otro de numeros aleatorios es decir del juego en si.

2. Utilizare la clase UI, para inicializar los componentes HTML y poder manejar cada uno de ellos desde ahi
